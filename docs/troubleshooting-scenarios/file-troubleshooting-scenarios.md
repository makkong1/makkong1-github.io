# File 도메인 트러블슈팅 시나리오

## 발생 가능한 문제 상황들

### 1. 파일 업로드 시 대용량 파일 처리 문제
- **시나리오**: 대용량 파일 업로드 시 메모리 부족 또는 타임아웃
- **재현 방법**: 대용량 파일 업로드 시도
- **예상 증상**: 메모리 부족, 타임아웃 발생

### 2. 파일 삭제 시 실제 파일 미삭제
- **시나리오**: 파일 삭제 시 DB 레코드는 삭제되었지만 실제 파일은 삭제되지 않음
- **재현 방법**: 파일 삭제 후 실제 파일 확인
- **예상 증상**: 디스크 공간 낭비

### 3. 파일 업로드 중복 저장
- **시나리오**: 같은 파일을 여러 번 업로드할 때 중복 저장
- **재현 방법**: 같은 파일을 여러 번 업로드
- **예상 증상**: 디스크 공간 낭비

### 4. 파일 다운로드 시 권한 검증 누락
- **시나리오**: 파일 다운로드 시 접근 권한 확인 없이 다운로드 가능
- **재현 방법**: 권한 없는 파일 다운로드 시도
- **예상 증상**: 권한 없이 다운로드 가능

### 5. 파일 목록 조회 N+1 문제
- **시나리오**: 파일 목록 조회 시 각 파일의 메타데이터를 개별 쿼리로 조회
- **재현 방법**: 파일 목록 조회 시 로그 확인
- **예상 증상**: 파일마다 메타데이터 조회 쿼리 발생

### 6. 파일 동기화 시 실제 파일 미존재
- **시나리오**: 파일 동기화 시 DB에는 있지만 실제 파일은 없음
- **재현 방법**: 파일 동기화 후 실제 파일 확인
- **예상 증상**: 동기화 실패

### 7. 파일 업로드 시 파일 타입 검증 누락
- **시나리오**: 파일 업로드 시 허용되지 않은 파일 타입 업로드 가능
- **재현 방법**: 허용되지 않은 파일 타입 업로드 시도
- **예상 증상**: 보안 문제

### 8. 파일 저장 경로 중복
- **시나리오**: 파일 저장 시 같은 경로에 파일 저장 시도
- **재현 방법**: 동시에 같은 경로에 파일 저장
- **예상 증상**: 파일 덮어쓰기 또는 저장 실패
