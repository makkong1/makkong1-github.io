# Report 도메인 트러블슈팅 시나리오

## 발생 가능한 문제 상황들

### 1. 중복 신고 처리
- **시나리오**: 같은 사용자가 같은 대상에 대해 여러 번 신고 가능
- **재현 방법**: 같은 사용자가 같은 대상에 대해 여러 번 신고 시도
- **예상 증상**: 중복 신고 생성

### 2. 신고 처리 시 자동 제재 적용 실패
- **시나리오**: 신고 처리 시 자동 제재 적용 중 예외 발생
- **재현 방법**: 제재 적용 중 예외 발생
- **예상 증상**: 신고는 처리되었지만 제재는 적용되지 않음

### 3. 신고 목록 조회 성능 저하
- **시나리오**: 관리자가 신고 목록 조회 시 각 신고의 신고자, 처리자 정보를 개별 쿼리로 조회
- **재현 방법**: 신고 목록 조회 시 로그 확인
- **예상 증상**: 신고마다 연관 데이터 조회 쿼리 발생

### 4. 신고 처리 시 상태 업데이트 동시성 문제
- **시나리오**: 여러 관리자가 동시에 같은 신고 처리 시도
- **재현 방법**: 동시에 신고 처리 API 호출
- **예상 증상**: 상태 업데이트 불일치

### 5. 신고 대상 타입 검증 누락
- **시나리오**: 신고 대상 타입이 실제로 존재하는지 확인하지 않음
- **재현 방법**: 존재하지 않는 대상 타입으로 신고 시도
- **예상 증상**: 잘못된 신고 생성 가능

### 6. 신고 처리 시 관리자 권한 검증 누락
- **시나리오**: 신고 처리 시 관리자 권한 확인 없이 처리 가능
- **재현 방법**: 일반 사용자가 신고 처리 시도
- **예상 증상**: 권한 없이 신고 처리 가능

### 7. 신고 자동 제재 로직 누락
- **시나리오**: 신고 처리 시 자동 제재 로직이 없어 수동으로만 제재 가능
- **재현 방법**: 신고 처리 후 제재 확인
- **예상 증상**: 자동 제재 미적용

### 8. 신고 통계 조회 성능 저하
- **시나리오**: 신고 통계 조회 시 매번 집계 쿼리 실행
- **재현 방법**: 신고 통계 조회 API 호출 시 쿼리 확인
- **예상 증상**: 매번 집계 쿼리 발생, 응답 시간 증가
