# Petory 프로젝트 PDF 생성 가이드

## 🚀 추천 방법별 상세 가이드

### **방법 1: 브라우저 인쇄 기능 (가장 빠름) ⭐ 추천**

#### 단계:
1. 구성안 내용을 HTML 파일로 만들기
2. 브라우저에서 열기
3. `Ctrl + P` (Windows) 또는 `Cmd + P` (Mac)
4. PDF로 저장

#### 장점:
- 즉시 사용 가능
- 무료
- 별도 도구 설치 불필요

#### 단점:
- 디자인 커스터마이징 제한적
- 다이어그램은 수동 삽입 필요

---

### **방법 2: Canva (디자인 품질 높음) ⭐⭐ 추천**

#### 단계:
1. [Canva](https://www.canva.com) 회원가입 (무료)
2. "프레젠테이션" 또는 "문서" 템플릿 선택
3. 구성안 내용 입력
4. 디자인 커스터마이징
5. 다운로드 → PDF로 저장

#### 장점:
- 전문적인 디자인
- 템플릿 활용 가능
- 아이콘, 이미지 삽입 용이
- 무료 플랜으로도 충분

#### 단점:
- 수동 입력 필요
- 시간 소요

---

### **방법 3: Marp (개발자 친화적) ⭐⭐⭐ 강력 추천**

#### 설치:
```bash
# VS Code 확장 설치
# 1. VS Code에서 확장 검색: "Marp for VS Code"
# 2. 설치 후 .md 파일에서 Marp 프리뷰 실행
```

#### 사용법:
1. 구성안을 Marp 형식으로 변환
2. VS Code에서 열기
3. Marp 프리뷰 보기
4. PDF로 Export

#### 장점:
- 마크다운 기반 (코드로 관리)
- 자동 페이지 분할
- Mermaid 다이어그램 지원 가능
- 버전 관리 가능 (Git)

#### 단점:
- 학습 곡선 (하지만 쉽음)

---

### **방법 4: Notion (간단한 문서 작성) ⭐⭐**

#### 단계:
1. [Notion](https://www.notion.so) 무료 계정 생성
2. 새 페이지 생성
3. 구성안 내용 입력/복사
4. 디자인 꾸미기
5. 메뉴 → Export → PDF

#### 장점:
- 마크다운 지원
- 직관적인 에디터
- 무료

#### 단점:
- 다이어그램은 수동 삽입
- 디자인 자유도 제한적

## 💡 추가 팁

- **다이어그램 처리**: Mermaid 다이어그램은 이미지로 변환 필요
  - [Mermaid Live Editor](https://mermaid.live)에서 PNG/SVG Export

- **일관된 디자인**: 색상 팔레트 미리 정하기
  - 주 색상: #ff7e36 (포트폴리오 테마)
  - 배경, 텍스트 색상도 미리 결정

- **폰트**: 가독성 좋은 폰트 선택
  - 제목: 굵은 폰트
  - 본문: 명확한 폰트 (예: Noto Sans, Pretendard)
