# Care 도메인 예외처리 리팩토링

## 1. 현황 분석

### 1.1 예외 발생 위치 및 유형

| 서비스/컨트롤러 | 예외 메시지 | 현재 타입 | 권장 HTTP |
|----------------|-------------|-----------|-----------|
| **CareRequestController** | 인증되지 않은 사용자입니다 | RuntimeException | 401 |
| | 유효하지 않은 사용자 ID입니다 | RuntimeException | 400 |
| **CareRequestService** | CareRequest not found | RuntimeException | 404 |
| | User not found | RuntimeException | 404 |
| | 이메일 인증이 필요합니다 | EmailVerificationRequiredException (user) | 403 |
| | 제공 코인은 0보다 커야 합니다 | RuntimeException | 400 |
| | 펫코인 잔액이 부족합니다 | RuntimeException | 400 |
| | Pet not found | RuntimeException | 404 |
| | 펫 소유자만 펫 정보를 연결할 수 있습니다 | RuntimeException | 403 |
| | 본인의 케어 요청만 수정할 수 있습니다 | RuntimeException | 403 |
| | 본인의 케어 요청만 삭제할 수 있습니다 | RuntimeException | 403 |
| | 작성자 또는 승인된 제공자만 상태를 변경할 수 있습니다 | RuntimeException | 403 |
| | 코인 지급 처리 중 오류가 발생했습니다 | RuntimeException | 500 |
| | 환불 처리 중 오류가 발생했습니다 | RuntimeException | 500 |
| **CareRequestCommentService** | CareRequest not found | IllegalArgumentException | 404 |
| | User not found | IllegalArgumentException | 404 |
| | 당신은 댓글 작성 불가입니다 | IllegalStateException | 403 |
| | Comment not found | IllegalArgumentException | 404 |
| | Comment does not belong to the specified care request | IllegalArgumentException | 400 |
| **CareReviewService** | CareApplication ID가 필요합니다 | IllegalArgumentException | 400 |
| | CareApplication not found | RuntimeException | 404 |
| | 승인된 펫케어 서비스에만 리뷰를 작성할 수 있습니다 | IllegalStateException | 409 |
| | 이미 해당 서비스에 리뷰를 작성하셨습니다 | IllegalStateException | 409 |
| | 요청자만 리뷰를 작성할 수 있습니다 | IllegalArgumentException | 403 |
| | 제공자에게만 리뷰를 작성할 수 있습니다 | IllegalArgumentException | 403 |
| | Reviewer not found | RuntimeException | 404 |
| | Reviewee not found | RuntimeException | 404 |
| **CareRequestScheduler** | catch (Exception e) | - | 스케줄러 로그만 |

### 1.2 문제점
1. **예외 타입 혼용**: RuntimeException vs IllegalArgumentException vs IllegalStateException
2. **메시지 혼재**: "CareRequest not found" vs "CareApplication not found" (영어)
3. **User/Pet 도메인 의존**: UserNotFoundException, PetNotFoundException, EmailVerificationRequiredException 재사용 필요
4. **권한/상태 검증**: "본인만", "요청자만", "승인된 서비스에만" 등 → CareForbiddenException 또는 CareValidationException

## 2. 도입할 예외 클래스

### 2.1 구조
```
domain/care/exception/
├── CareRequestNotFoundException.java   # 404 - 펫케어 요청 미존재
├── CareApplicationNotFoundException.java # 404 - 펫케어 신청 미존재
├── CareCommentNotFoundException.java   # 404 - 댓글 미존재
├── CareCommentNotBelongException.java   # 400 - 댓글이 해당 요청에 속하지 않음
├── CareForbiddenException.java          # 403 - 본인만 수정/삭제, 요청자/제공자만 리뷰 등
├── CareValidationException.java         # 400 - 제공 코인, CareApplication ID, 유효하지 않은 사용자 ID
├── CareConflictException.java           # 409 - 이미 리뷰 작성, 승인된 서비스에만 리뷰 등
└── CarePaymentException.java            # 500 - 코인 지급/환불 처리 오류 (선택)
```

### 2.2 User/Pet 도메인 예외 재사용
- **UnauthenticatedException**: 인증되지 않은 사용자 (CareRequestController)
- **UserNotFoundException**: User, Reviewer, Reviewee 조회 실패 시
- **PetNotFoundException**: Pet 조회 실패 시
- **EmailVerificationRequiredException**: 이메일 인증 필요 시

### 2.3 ApiException 베이스
- `global/exception/ApiException.java` 상속 (기존 유지)

## 4. 예외 정적 메서드 제안

### CareForbiddenException
```java
CareForbiddenException.ownRequestOnly()           // 본인의 케어 요청만 수정/삭제할 수 있습니다
CareForbiddenException.ownerOrApprovedProvider() // 작성자 또는 승인된 제공자만 상태를 변경할 수 있습니다
CareForbiddenException.requesterOnly()            // 요청자만 리뷰를 작성할 수 있습니다
CareForbiddenException.providerOnly()             // 제공자에게만 리뷰를 작성할 수 있습니다
CareForbiddenException.commentNotAllowed()        // 당신은 댓글 작성 불가입니다
CareForbiddenException.petOwnerOnly()             // 펫 소유자만 펫 정보를 연결할 수 있습니다
```

### CareValidationException
```java
CareValidationException.careApplicationIdRequired()  // CareApplication ID가 필요합니다
CareValidationException.coinMustBePositive()         // 제공 코인은 0보다 커야 합니다
CareValidationException.insufficientBalance()        // 펫코인 잔액이 부족합니다
CareValidationException.invalidUserId()              // 유효하지 않은 사용자 ID입니다
```

### CareConflictException
```java
CareConflictException.approvedServiceOnly()   // 승인된 펫케어 서비스에만 리뷰를 작성할 수 있습니다
CareConflictException.alreadyReviewed()       // 이미 해당 서비스에 리뷰를 작성하셨습니다
```

### CarePaymentException (선택)
```java
CarePaymentException.paymentFailed(Throwable)  // 코인 지급 처리 중 오류가 발생했습니다
CarePaymentException.refundFailed(Throwable)   // 환불 처리 중 오류가 발생했습니다
```
